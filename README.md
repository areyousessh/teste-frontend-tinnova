# Desafio Tecnico Frontend Tinnova

Desafio Técnico criado para o processo de candidatura para vaga de Desenvolvedor Fullstack pleno na empresa Tinnova utilizando NodeJS e React

### Desafio 

Foi pedido no desafio a criação de um cadastrado de usuários utilizando react, algum processador de css e que os dados ficassem armazenados no localhost

### Tecnologias utilizadas
- React
- Typescript
- Tailwind
- React Hook Form 
- Yup

### Como utilizar o projeto

Já tendo clonado esse repositório você precisará realizar os seguintes passos

- Execute um ```yarn```para baixar as dependencias e em seguida execute um ```yarn dev```para rodar a aplicação
